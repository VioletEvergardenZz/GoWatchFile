## 6. 维护动作

1. 新增文档时，必须在本清单登记状态、用途和最近校准日期。
2. 主线文档变更后，需同步更新：
   - `docs/README.md`
   - `docs/文档导航.md`
   - `docs/98-历史兼容/旧文档映射.md`
3. 涉及 API 契约或接口行为变更时，需按检查清单逐项核对：
   - `docs/00-文档治理/接口变更文档同步检查清单.md`
4. 组件 README（`go-watch-file/README.md`、`console-frontend/README.md`）更新后，需同步检查主线文档是否需要回链。
5. 删除或迁移兼容目录前，必须确认脚本、报告和外部引用已完成替换。
